# Голосовой помощник

Голосовой помощник на **Node.js/TypeScript** с API ChatGPT и **локальными** STT (распознавание речи) и TTS (синтез речи) без облачных сервисов для голоса.

## Возможности

- **Локальный STT** — распознавание речи через [echogarden](https://github.com/echogarden-project/echogarden) (движок Whisper, работает офлайн).
- **Локальный TTS** — синтез речи через echogarden (движок Kokoro).
- **ChatGPT** — ответы генерирует OpenAI API (нужен ключ).
- Режимы: запись с микрофона или ввод текста с клавиатуры.

## Требования

- **Node.js** 18+
- **SoX** — для записи с микрофона:
  - macOS: `brew install sox`
  - Linux: `sudo apt install sox libsox-fmt-all`
- **OpenAI API key** — для ChatGPT.

## Установка

```bash
cd voice-assistant
yarn install
cp .env.example .env
# Отредактируйте .env и укажите OPENAI_API_KEY=sk-...
```

## Запуск

```bash
yarn build
yarn start
```

Или одной командой:

```bash
yarn assist
```

## Использование

1. **1** — записать голос (нужен микрофон и SoX). Говорите, затем нажмите Enter, чтобы остановить запись.
2. **2** — ввести текст с клавиатуры (удобно для проверки без микрофона).
3. **q** — выход.

После вашего сообщения помощник отправит его в ChatGPT, выведет ответ в консоль и озвучит его локальным TTS.

## Переменные окружения (.env)

| Переменная       | Описание                          |
|------------------|-----------------------------------|
| `OPENAI_API_KEY` | Ключ API OpenAI (обязательно)     |
| `OPENAI_MODEL`   | Модель (по умолчанию: gpt-4o-mini)|

## Структура проекта

- `src/index.ts` — главный цикл, запись, меню.
- `src/stt.ts` — распознавание речи (echogarden).
- `src/tts.ts` — синтез речи и воспроизведение (echogarden + afplay/aplay).
- `src/chat.ts` — запросы к ChatGPT (OpenAI API).

## Воспроизведение ответа

На macOS используется `afplay`, на Linux — `aplay`. Убедитесь, что устройство вывода звука выбрано в системе.
